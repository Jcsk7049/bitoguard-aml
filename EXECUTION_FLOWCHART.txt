╚════════════════════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────────────────┐
│ 【前置準備】                                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. 檢查 Python 版本 (3.8+)                                                   │
│ 2. 安裝依賴套件 (xgboost, pandas, numpy, shap, boto3)                       │
│ 3. 準備資料檔案 (./data/ 目錄下的 CSV 檔案)                                 │
│ 4. 設置環境變數 (可選)                                                       │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【選擇執行方案】                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ┌─────────────────────┐  ┌──────────────────┐  ┌──────────────────────┐  │
│  │  方案 A             │  │  方案 B          │  │  方案 C              │  │
│  │  完整端到端         │  │  快速推論        │  │  分階段執行          │  │
│  │  (首次推薦)         │  │  (已有模型)      │  │  (調試用)            │  │
│  │                     │  │                  │  │                      │  │
│  │ --start-from        │  │ --only xai       │  │ --only <stage>       │  │
│  │ feature             │  │                  │  │                      │  │
│  │                     │  │ ⏱️ 3-5 秒        │  │ ⏱️ 依階段而定        │  │
│  │ ⏱️ 5-10 分鐘       │  │                  │  │                      │  │
│  └─────────────────────┘  └──────────────────┘  └──────────────────────┘  │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 1: 特徵工程】                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: ./data/ (CSV 檔案)                                                     │
│ 處理:                                                                        │
│   • 載入 6 個資料表 (users, twd_transfer, crypto_transfer, trades, labels)  │
│   • 計算 3 個特徵:                                                           │
│     - twd_deposit_count (台幣入金次數)                                      │
│     - twd_withdraw_count (台幣出金次數)                                     │
│     - night_tx_ratio (深夜交易比例)                                         │
│   • 合併訓練標籤和預測標籤                                                   │
│ 輸出: feature_cache.parquet (51,017 訓練 + 12,753 預測)                     │
│ ⏱️ 時間: 3-5 秒                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 2: 模型訓練】                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: feature_cache.parquet                                                 │
│ 超參數:                                                                      │
│   • max_depth: 8 (樹的最大深度)                                             │
│   • eta: 0.05 (學習率)                                                      │
│   • num_round: 1000 (迭代次數)                                              │
│   • scale_pos_weight: 30.1 (正負樣本比例)                                   │
│ 處理:                                                                        │
│   • 5-Fold 交叉驗證                                                          │
│   • 計算 Precision, Recall, F1 Score                                        │
│   • 訓練完整模型                                                             │
│ 輸出: model.json, cv_report.json                                            │
│ ⏱️ 時間: 2-3 分鐘                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 3: 下載模型】(可選，本地環境跳過)                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: SageMaker Training Job                                                │
│ 處理: 從 S3 下載模型                                                         │
│ 輸出: model.json                                                            │
│ ⏱️ 時間: 1-2 分鐘                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 4: 驗證報告】(可選)                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: model.json, feature_cache.parquet                                     │
│ 處理:                                                                        │
│   • Threshold Sweep (0.3 ~ 0.9)                                             │
│   • 計算最佳決策閾值                                                         │
│   • 生成技術報告                                                             │
│ 輸出: validation_report.json                                                │
│ ⏱️ 時間: 1-2 分鐘                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 5: 視覺化】(可選)                                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: model.json, feature_cache.parquet                                     │
│ 處理:                                                                        │
│   • PR 曲線 (Precision-Recall)                                              │
│   • SHAP Beeswarm 圖                                                        │
│   • 特徵重要性排名                                                           │
│   • 混淆矩陣                                                                 │
│ 輸出: plots/ 目錄 (4 張圖表)                                                │
│ ⏱️ 時間: 30-60 秒                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【Stage 6: XAI 診斷 + 提交】                                                │
├─────────────────────────────────────────────────────────────────────────────┤
│ 輸入: model.json, feature_cache.parquet                                     │
│ 處理:                                                                        │
│   • 生成預測機率 (12,753 個用戶)                                            │
│   • 計算 SHAP 特徵貢獻度                                                     │
│   • 調用 Bedrock Claude 生成診斷書                                          │
│   • 分級風險 (EXTREME, HIGH, BOUNDARY, MEDIUM)                             │
│   • 生成風控建議                                                             │
│ 輸出:                                                                        │
│   • submission.csv (user_id, status)                                        │
│   • submission_with_prob.csv (含預測機率)                                   │
│   • xai_reports.json (詳細診斷報告)                                         │
│ ⏱️ 時間: 3-5 秒                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【驗證結果】                                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. 檢查 submission.csv 格式                                                 │
│    head -5 submission.csv                                                   │
│                                                                              │
│ 2. 統計黑名單用戶                                                            │
│    python -c "import pandas as pd; df = pd.read_csv('submission.csv');     │
│    print(f'黑名單: {(df[\"status\"]==1).sum()}')"                           │
│                                                                              │
│ 3. 檢查診斷報告                                                              │
│    cat xai_reports.json | python -m json.tool                              │
│                                                                              │
│ 4. 檢查模型性能                                                              │
│    python verify_output.py                                                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│ 【提交競賽】                                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│ 上傳檔案: submission.csv                                                    │
│ 到競賽平台                                                                   │
└─────────────────────────────────────────────────────────────────────────────┘
